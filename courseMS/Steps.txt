1. Создали приложение с микросервисом (проект coursems)
2. Проверили работу этого сервиса с локальной БД(http://localhost:8080/api/course
3. Создаем jar файл приложение с пом. Maven (на проекте прав.мыши run as-Maven install
4.Создаем Dockerfile который описывает процесс сборки контейнера с нашим приложением
5. Собираем контейнер с нашим микросервисом командой docker build .
6. Описываем группу контейнеров для работы нашего приложения в файле docker-compose.yml
7. Останавливаем локальные сервера (веб сервер в эклипсе и сервер mysql)
 8. Запускаем группу контейнеров командой docker-compose up
